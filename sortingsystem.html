Object-Sorting Conveyor System (Mechatronics)

Tools: AVR C • ADC + interrupts • Reflectivity sensor • Stepper motor (S-curve profile) • DC motor (PWM belt) • Testing & calibration

Problem

Design and build an automated conveyor that detects objects using a reflectivity sensor and sorts them into bins using a stepper-driven diverter. The main constraints were speed (finish the sorting task quickly) and reliability (avoid missed detections and stepper slipping at higher throughput).

Approach

  • Built an interrupt-driven detection pipeline in AVR C using ADC sampling to capture reflectivity changes as objects pass the sensor.
  • Implemented a queue-based timing/scheduling approach so the diverter could actuate at the correct moment relative to belt motion.
  • Tuned belt speed using PWM; identified ~70% duty cycle as the best tradeoff between throughput and stepper traction (higher caused slipping).
  • Developed S-curve acceleration profiles (delay arrays) for the stepper to reduce shock loading and prevent missed steps during 90° and 180° moves.
  • Iterated through calibration runs to set detection thresholds, validate timing offsets, and confirm repeatability across multiple trials.

Results

  • Achieved a stable “fast” configuration with belt PWM ~70% and reliable stepper actuation during continuous operation.
  • Stepper profiles: 90° turn used 50 delay entries totaling ~0.535 s; 180° turn used 100 delay entries totaling ~0.78 s.
  • Demonstrated system performance of SPI = 48 / 33 s = 1.45 using the tested scoring method.

What I’d improve next

  • Add additional algorithms such as a pre-turn algorithm to make the stepper motor turn before the first piece in the queue reached the end gate to sort faster.
  • Improve the code effeciency to allow both the end gate sensor and reflectivity sensor to operate at the same time.
  • Improve thermal management (heatsink/airflow or current limiting) and add firmware temperature/time-based derating to prevent overheating-related slip of the stepper motor.
  • Optimize the s-curve for the stepper motor to make it turn faster without slipping. This would also allow an increase in the PWM belt speed.
